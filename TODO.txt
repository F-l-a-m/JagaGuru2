Список заданий для Java 2
Репозиторий: https://github.com/F-l-a-m/JavaGuru2

Краткое описание задачи: создать чат с поддержкой создания, удаления, редактирования комнат и пользователей.
Опебпечить чат возможностью регистрации пользователей.

* * * * * * * * * *
* Основные задачи *
* * * * * * * * * *

+ Консольный чат с индусским кодом
+ Консольный чат MVC
+ Создать простой чат без поддержки базы данных и регистрации
+ Команды для управления чатом, например /quit, /nick, /join ...
    Созданы:
        /nick nickname,
        /quit,
        /r - обновить список сообщений в комнате (обновить консоль),
        /join roomname,
        /leave roomname,
        /list - вывод списка всех комнат чата.
+ Служба управления пользователями
+ Служба управления сообщениями
+ Служба управления комнатами чата
+ Валидация смены ника
+ Тесты для валидатора смены ников
+ Тесты для UserService, пока что только для ChangeNickname метода
+ Тесты для обрабочтика ввода пользователя (HandleChatInputService)
+ Возможность создания дополнительных комнат
+ Возможность перехода по комнатам чата
+ Хранение состояния чата в БД (список комнат и история сообщений в них)
о Администратор (создатель) чат комнаты
    o Возможность редактирования комнат чата
    o Возможность удаления комнат чата
        o автоматическое удаление при выходе последнего пользователя (для временной чат комнаты)
        о ручное удаление для постоянной чат комнаты, выкинуть всех при удалении ?
o Тесты для всех методов UserService
о Приветствие чат комнаты (заголовок) CRUD
о Постоянные и временные чат комнаты
